# AGENT.md（AI/Codex作業ルール）

## 目的
- 安全に、最小変更で、再現性ある開発支援を行う
- 破壊的操作・外部送信・勝手なGit操作を防ぐ

---

## ✅ 必ず守る（MUST）
- 必ず日本語で回答する
- コマンド実行前に「何をするか」を1〜2行で説明する
- 変更は小さく分けて進める（まとめて大量変更しない）
- 迷ったら実行せず質問する
- 作業の節目で `git status` と `git diff` を確認する

---

## ❌ 絶対に勝手にやらない（MUST NOT）
### Git操作
- **許可なく `git commit` しない**
- **許可なく `git push` しない**
- **`git push -f`（force push）は禁止**

### 破壊的コマンド
- **確認なしに破壊的・不可逆なコマンドを実行しない**
  - 例：`rm -rf` / `sudo rm -rf` / 危険な `mv` / `chmod -R` / `chown -R`
  - 例：`diskutil` / フォーマット / パーティション操作

### 外部送信
- 許可なく外部に情報を送らない（貼り付け/アップロード/公開）
- 許可なく未知のパッケージをグローバルインストールしない

---

## 🔥 強い一文（暴走防止・最重要）
- **破壊的または不可逆な操作は、実行前に必ず確認を取り、私が「OK」と言うまで実行しない。**
- **`git push` は私が明示的に「push OK」と言った場合のみ実行してよい。**

---

## ✅ 推奨する進め方（安全な手順）
1. 現状確認：`pwd` / `ls`
