# ymgnのdotfiles
.で始まるファイルやディレクトリを管理するRepository

dotfilesのファイルを変更した場合はcommitし、新規で追加する場合は合わせて ```dotfilesdotfilesLink.sh```にシンボリックリンクのコマンドを追加する

## 使い方
``` sh ~/dotfiles/dotfilesLink.sh ```
